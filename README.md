# Gizem & Osman Düğün Davetiyesi

Tek sayfalık statik davetiye. `index.html` dosyası doğrudan tarayıcıda açılabilir veya GitHub Pages, Netlify Drop, Vercel gibi statik hosting servislerinde yayınlanabilir.

## Yayınlama

1. Bu klasörü bir GitHub reposuna yükleyin.
2. GitHub'da `Settings > Pages` bölümünden branch olarak `main`, klasör olarak `/root` seçin.
3. Alternatif olarak klasörü Netlify Drop'a sürükleyip bırakabilirsiniz.

## Düzenleme

- Kapak arka planı: `assets/cover-cartoon.png`
- Kapak davetiye animasyonu ve parıltılar CSS ile oluşturuldu.
- Metin ve bağlantılar: `index.html`
- Görsel tasarım: `styles.css`
- Açılış, paylaşım ve geri sayım: `script.js`
